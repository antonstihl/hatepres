HTML-slides

The world's best documented presentation tool

A presentation library

HTML-slides provides simple tooling to render semantic HTML as a delightful visual aid for your presentation.

Why?

If you can code it, why not?

How to

  1. Create an .html file with your semantic HTML content, divided into sections.
  2. Wrap all sections (slides) in a div with id html-slides 
    <div id="html-slides">
  <section>...<section>
  <section>...<section>
</div>
  3. Add the head style tag
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/antonstihl/html-slides/styles.css" />
  4. Add the head script tag, deferred
    <script defer src="https://cdn.jsdelivr.net/gh/antonstihl/html-slides/script.js"></script>
  5. (Optional) add an indicator element, <pre id="indicator"></pre>
  6. Add class f (focusable) to make elements focusable. This will hide them initially, by default.
  7. Done! ‚úîÔ∏è

Bonus

Hook into the classes used by HTML-slides to style your elements as your presentation progresses.

  • current-section
  • current-focus
  • previous-focus

Philosophy

Some of the key characteristics of HTML-slides come from David JP Phillips TEDx talk, How to avoid death by PowerPoint.

    • One message per slide
    • Use contrast and size to direct attention
    • Don't write sentences
    • Dark bakground
    • Maximum of 6 items per slide

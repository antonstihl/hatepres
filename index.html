HTML-slides

The world's best documented presentation tool

A presentation library

HTML-slides provides simple tooling to render semantic-enough HTML as a delightful visual aid for your presentation.

Why?

If you can code a presentation, why wouldn't you?

How to

  1. Create an .html file with your HTML content, divided into one <section/> per "slide".
  2. Add the head style tag <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/antonstihl/html-slides/styles.css" />
  3. Add the head script tag <script src="https://cdn.jsdelivr.net/gh/antonstihl/html-slides/script.js"></script>
  4. Add attribute hs-f (f for focusable) to make elements focusable. This will hide them and reveal on focus.
    • Bonus: Add a CSS selector as the hs-f value if you want to...

Bonus

Hook into the classes/attributes used by HTML-slides to
style your elements as your presentation progresses.

  • [hs-f]
  • [hs-fa] (anchored focus, not focusable on its own)
  • .p (perennial, used to escape [hs-f]'s default hiding)
  • section.current
  • .current-focus
  • .focus-trace

Philosophy

Some of the key characteristics of HTML-slides come from David JP Phillips TEDx talk, How to avoid death by PowerPoint.

    • One message per slide
    • Use contrast and size to direct attention
    • Don't write sentences
    • Dark bakground
    • Maximum of 6 items per slide
